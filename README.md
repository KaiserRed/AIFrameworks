# Лабораторные по прикладным системам и фреймворкам искусственного интеллекта

## Датасеты

#### Классификация
`Cancer_Data.csv` (Cacer Classification — классификация рака). [Ссылка](https://www.kaggle.com/datasets/erdemtaha/cancer-data/data?select=Cancer_Data.csv). 

**Практическая задача:** Автоматическая диагностика рака молочной железы на основе цитологических характеристик клеток, полученных методом тонкоигольной аспирации.

#### Регрессия
`laptop_prices.csv` (Laptop Prices — предсказание цен на ноутбуки). [Ссылка](https://www.kaggle.com/datasets/owm4096/laptop-prices).

Практическая задача:
Применение в e-commerce для автоматического ценообразования, анализа рынка и рекомендательных систем.


## Ноутбуки

- `knn.ipynb` — KNN (классификация + регрессия)
- `regression.ipynb` — логистическая и линейная регрессии
- `decision_tree.ipynb` — решающее дерево
- `random_forest.ipynb` — случайный лес
- `grad_boosting.ipynb` — градиентный бустинг
